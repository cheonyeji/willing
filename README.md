# 윌링

효율적인 할 일 관리를 위해 아날로그, 온라인 방식 등 다양한 방식을 사용해 보는 과정에서 필요한 기능을 모두 갖춘 서비스를 개발하고자 하는 욕심이 생겼습니다. 따라서 “하지 못할 이유가 없는, 기꺼이 하는"이라는 사전적 의미를 가진 단어 윌링(Willing)이란 이름으로 서비스를 기획 및 개발하였습니다.

서비스는 현재 https://willing.vercel.app/ 에서 이용 가능합니다.

# 개발 환경

- React.js + Typescript
- styled-components
- Recoil
